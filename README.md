## What Is an Industrial Ethernet Switch?

An industrial ethernet switch is a network device designed to support harsh environmental conditions while delivering high-performance Ethernet connectivity. Versitron’s unmanaged switch offers industrial switch system overview, advanced industrial ethernet switch hardware features, and adherence to industrial switches manufacturing standards, making it ideal for real-world industrial applications.

### Key Features

- 8 × 10/100 Mbps copper RJ45 ports  
- 2 × 100 Mbps multimode fiber ports (ST connector)  
- Rugged, DIN rail or panel mountable enclosure  
- Supports IEEE 802.3, 802.3u, 100Base-FX standards  
- Flow control: IEEE 802.3x (full-duplex), back-pressure (half-duplex)  
- Broadcast storm protection  
- Alarm relay output for power failure  
- LED indicators for power and port status  
- Operating temperature: −30 °C to +70 °C  
- Relative humidity: 5 % to 95 % non-condensing
